# 🎉 Multi-feature Nandu App

A Python-based console application that combines multiple everyday functionalities into a single app.

## 🚀 Features

### 🛒 Groceries Shopping
- Add items from categories like Dresses, Sarees, Jewellery, Electronics, etc.
- Select colors, types, and models
- Generates total bill

### 🍕 Food Ordering
- Order items like Pizza, Burger, Samosa, Tea, Ice-Cream, Chocolates
- Supports quantity selection
- Displays final food bill

### 💰 Money Transfer
- Secure transfer using PIN
- Includes sender name and timestamp

### 💳 Bill Payments
- Pay bills using:
  - PhonePe
  - Paytm
  - Google Pay
  - Amazon Pay
- Includes timestamp and PIN authentication

### 🎬 Movies & 🎵 Songs
- Displays latest movie list
- Displays trending songs list
